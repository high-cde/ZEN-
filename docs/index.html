Cos'è ZEN?

ZEN è un ecosistema modulare composto da linguaggio, runtime, OS, AI interna, filesystem virtuale, database nativo e servizi autonomi. Ogni componente si costruisce, si ripara e si ottimizza da solo.

Moduli Principali

zVM

Virtual Machine stack-based per esecuzione bytecode ZEN.

zASM

Assembler e disassembler ufficiale del linguaggio.

zNET

Networking astratto, mesh e message passing.

zSEC

Sandbox, permessi granulari e sicurezza del runtime.

zBOOT

Bootloader del micro‑OS GENESIS.

Moduli Avanzati

zAI

Motore AI interno: analisi AST, ottimizzazione e auto‑documentazione.

zDB

Database nativo key-value e document store.

zFS

Filesystem virtuale in memoria.

zUI

Toolkit UI astratto con widget e layout engine.

zOS — ZEN Operating System

Kernel modulare, scheduler, memory manager, process manager, device manager, shell e init system. Completato da servizi autonomi: logger, networkd, timed, eventd e registry.
